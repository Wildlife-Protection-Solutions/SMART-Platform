SMART Platform

Relationship to SERCA Desktop
----

[SERCA Desktop](https://github.com/Wildlife-Protection-Solutions/SERCA-Desktop) is the planned successor to SMART Desktop. Until SERCA Desktop is released and migration is complete, this repository remains the active source for SMART Desktop/Connect and may continue to receive bug fixes and other changes.

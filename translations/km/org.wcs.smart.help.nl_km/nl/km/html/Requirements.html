System Requirements

Hardware / OS Requirements

The software operates under the following operating systems and minimum memory configurations:

  • Windows XP: 1 GB of RAM
  • Windows 7: 2 GB of RAM
  • Linux, Ubuntu and Xubuntu (9.10): 1 GB of RAM
  • Mac OS/X, Version 10.6 or 10.7: 2 GB of RAM

Performance will vary depending on the amount of memory available and the processor speed. The application will be slower on computers with single core processors that meet the minimum requirements, and will perform better on computers that exceed these requirements. Netbooks are not recommended.

Touch-screen based devices (such as smartphones and tablets) are not supported. In the future, data-collection applications may be developed for such devices, however the SMART application will not run on these devices.

Free Disk Space

The application requires a minimum of 2 GB of free disk space. However, depending on the data captured, the actual disk space requirements will vary widely between sites. The application will require sufficient disk space to store:

  • SMART software
  • SMART database
  • Basemap data - Vector and Imagery data
  • Photos and Videos associated with observations

USB / External Media Transfers

Database extracts are often shared between sites using USB keys or other external media. These devices will need to be of appropriate size to support the Conservation Area databases and external media files. When exporting databases and other objects, there is an option to include or exclude media files if appropriate.

Screen Resolution

The application has been developed for a minimum screen resolution of 1024 x 768.

Internet Connection

It is understood that although some installations will have full Internet access, many may be connected to the Internet only occasionally, and some may not be connected at all.

The implications are two-fold:

  • Use of one or more WMS services to supply basemap and map layer data will be available only if Internet access exists. Without access, such data must be provided on the local machine (as vector or raster files) and will be accessed directly (no WMS required).
  • Use of email to send database extracts as email attachments will be practical only with Internet access. Otherwise, USB sticks (most common), USB drives or other hard media will be used.

Application Installation

  • If you have a zipped application file (on CD, DVD or flash drive), copy the appropriate Windows (win32) or Linux (linux) zipped file to a local folder and unzip (extract the files).
  • If you have the application files (on CD, DVD or flash drive), copy either the Windows (win32) or Linux (linux) directory to your local hard drive.
  • Run the executable SMART.exe

Requisitos de sistema

Requisitos de Hardware/SO

O software opera sob os seguintes sistemas operacionais e configurações mínimas de memória:

  • Windows XP: 1 GB de RAM
  • Windows 7: 2 GB de RAM
  • Linux, Ubuntu e Xubuntu (9.10): 1 GB de RAM
  • Mac OS/X, versão 10.6 ou 10.7: 2 GB de RAM

O desempenho irá variar dependendo da quantidade de memória disponível e da velocidade do processador. O aplicativo será mais lento em computadores com processadores de núcleo único que atendam aos requisitos mínimos e funcionará melhor em computadores que excedam esses requisitos. Netbooks não são recomendados.

Dispositivos baseados na tela touchscreen (como smartphones e tablets) não suportam o aplicativo. No futuro, os aplicativos de coleta de dados podem ser desenvolvidos para esses dispositivos; no entanto, o aplicativo SMART não será executado nesses dispositivos.

Espaço livre em disco

O aplicativo requer um mínimo de 2 GB de espaço livre em disco. Mesmo assim, dependendo dos dados capturados, os requisitos reais de espaço em disco variam muito entre os sites. O aplicativo exigirá espaço em disco suficiente para armazenar:

  • Software SMART
  • Base de dados SMART
  • Dados do mapa de base - Dados vetoriais e de imagens
  • Fotos e Vídeos associados a observações

Transferências com mídia USB/externa

As extrações da base de dados costumam ser compartilhadas entre sites usando chaves USB ou outras mídias externas. Esses dispositivos precisarão ter tamanho adequado para suportar as bases de dados da Área de Conservação e os arquivos de mídia externos. Ao exportar bancos de dados e outros objetos, há uma opção para incluir ou excluir arquivos de mídia, se apropriado.

Resolução da tela

O aplicativo foi desenvolvido para uma resolução de tela mínima de 1024 x 768.

Conexão de internet

Entende-se que, embora algumas instalações tenham acesso total à Internet, muitas delas podem estar conectadas à Internet apenas ocasionalmente, e algumas podem não estar conectadas.

As implicações são duplas:

  • O uso de um ou mais serviços WMS para fornecer dados de mapa de base e de camada de mapa estará disponível somente se houver acesso à Internet. Sem acesso, esses dados devem ser fornecidos na máquina local (como arquivos vetoriais ou rasterizados) e serão acessados ​​diretamente (não é necessário WMS).
  • O uso de email para enviar extrações de banco de dados como anexos de email será prático somente com acesso à Internet. Caso contrário, os pendrives USB (mais comuns), drives USB ou outras mídias duras serão usados.

Instalação de aplicativos

  • Se você tiver um arquivo de aplicativo compactado em ZIP (em CD, DVD ou unidade flash), copie o arquivo compactado apropriado do Windows (win32) ou Linux (linux) para uma pasta local e descompacte (extraia os arquivos).
  • Se você tiver os arquivos do aplicativo (em CD, DVD ou unidade flash), copie o diretório Windows (win32) ou Linux (linux) para o disco rígido local.
  • Execute o arquivo executável SMART.exe

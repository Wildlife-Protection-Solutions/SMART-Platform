Requisitos del Sistema

Requisitos de Hardware / Sistema de Operación

El software funciona bajo los siguientes sistemas operativos y configuraciones de memoria mínimas:

  • Windows XP: 1 GB de RAM
  • Windows 7: 2 GB de RAM
  • Linux, Ubuntu y Xubuntu (versiones faltan determinar): 1 GB de RAM
  • Mac OS / X, versión 10.6 o 10.7: 2 GB de RAM

El rendimiento variará dependiendo de la cantidad de memoria disponible y la velocidad del procesador. La aplicación será más lenta en equipos con procesadores de un solo núcleo que cumplen los requisitos mínimos, y será más eficaz con mejores resultados en equipos que superen estos requisitos. No se recomiendan los Netbooks.

Maquinas con pantalla táctil (como teléfonos inteligentes y tabletas) no son soportadas. En el futuro, aplicaciones para colección de datos pueden ser desarrollados para este tipo de máquina, sin embargo, la aplicación SMART todavía no se puede ejecutar en estas máquinas.

Espacio libre en disco

La aplicación requiere un mínimo de 2 GB de espacio libre en el disco duro. Sin embargo, en función de los datos recoleccionados, los requisitos de espacio reales pueden variar ampliamente entre los sitios. La aplicación requiere suficiente espacio de disco duro para almacenar:

  • Software SMART
  • Base de datos de SMART
  • Datos para el mapa base-- datos vectoriales y datos de imagén
  • Fotos y vídeos asociados con las observaciones

Transferencias de medios USB/externos

Extractos de bases de datos son a menudo compartidas entre los sitios utilizando USBs u otros medios de comunicación externos. Estos aparatos necesitan ser de tamaño adecuado para soportar las bases de datos del área de conservación y archivos de medios externos. Al exportar bases de datos y otros objetos, hay una opción para incluir o excluir los archivos de media (como fotos u otros) si es apropiado.

Resolución de pantalla

La aplicación ha sido desarrollada para una resolución de pantalla mínima de 1024 x 768.

Conexión a Internet

Se entiende que, si bien algunas instalaciones tendrán acceso completo al Internet, muchos pueden estar conectados a Internet sólo ocasionalmente, y algunos pueden que no se puedan conectar en ningún momento.

Las implicaciones son de dos tipos:

  • El uso de uno o más servicios WMS para suministrar el mapa base y los datos de la capa de mapa solo serán disponibles si cuenta con acceso a Internet. Sin acceso, estos datos deben ser proveídos en el equipo local (como archivos vectoriales o de raster) y serán accedidos directamente (no se requiere WMS).
  • El uso del correo electrónico para enviar extractos de bases de datos como archivos adjuntos solo será práctico con acceso a Internet. Si no, las memorias USB (el más común) u otros medios de memoria física se usarán.

Instalación de la aplicación

  • Si usted tiene un archivo de aplicación en zip (en CD, DVD o unidad flash), copia el archivo zip apropiado para Windows (win32) o Linux (linux), a una carpeta local y descomprime (extraer los archivos).
  • Si usted tiene los archivos de la aplicación (en CD, DVD o unidad flash), copia el directorio apropiado Linux (linux) o Windows (win32) al disco duro local,
  • Ejecute el archivo ejecutable smart.exe

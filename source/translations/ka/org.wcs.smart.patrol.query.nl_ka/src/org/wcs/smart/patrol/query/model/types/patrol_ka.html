Patorl Query

Displays the patrols that match the given query filters. Filter options include all the patrol attributes (type, station, team etc), and data model categories and attributes.

Example Patrol Query Result

The results include one record per patrol leg that matches the query filters. Only patrol attributes are included in the results page. There is no information about observations or incidents included.

Details about the patrols are displayed in the tabular window. The patrol GPS tracks are displayed in the mapping window.

Survey Mission Track Query

Display all mission tracks that match the given filters. Filters include track type, sampling unit, mission and area and area applied to the track.

Mission Track Query vs Mission Query

Mission Track Queries generate on one record per track (and include track properties). Filters are limited to track, mission and survey properties (cannot query on observation details). Mission Queries generate on one record per mission and merge all tracks into a single feature on the map. Filters include mission and survey properties as well as observation details (data model filters).

Results will include one record per track that matches the given filters. No observation or incident information is included in the results. Track details are displayed in a tabular window. Tracks locations are displayed in a map window.

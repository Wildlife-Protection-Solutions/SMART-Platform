Usage:
    /? or /help or --help               - This message
    /silent                             - No console output
    /deleteAfterDownload                - Delete files that are successfully downloaded
    /download <sourceFilespec> <target> - Download data from attached device to target directory
    /upload <source> <target>           - Upload a file to a target

Examples:
    mtpcopy /deleteAfterDownload /download \mydata\*.json c:\mydata
    mtpcopy /upload c:\myapp\myapp.apk \myapp.apk

Note:
    1. Upload can only copy to the root of the device or one folder deep, i.e. you could use \myapp.apk, \ct\myapp.apk, but not \ct\sub\myapp.apk
    2. Process exit code will be 0 in case of success
    3. Other codes:
        ERROR_BAD_COMMAND_LINE = 100
        ERROR_DEVICE_NOT_FOUND = 101
        ERROR_DEVICE_NOT_CONNECTED = 102

 
Other Nodes:
If folder does not exist on device when writing; the folder will not be created and the write will fail.

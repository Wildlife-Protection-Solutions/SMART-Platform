Consulta em Grade de Entidade

Consultas de Entidade em Grade são as mesmas que as outras consultas em grade; no entanto, os dados também podem ser filtrados nas propriedades da entidade. Os resultados são exibidos em um formato de grade (gráfico raster) especificado pelo usuário com resultados agregados com base no tamanho de grade definido. Os resultados podem ser visualizados em uma tabela ou em um mapa.

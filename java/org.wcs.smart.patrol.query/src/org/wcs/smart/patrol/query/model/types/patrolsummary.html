A summary summarizes the raw data and allows for grouping into different categories. Items that can be summarized are values such as total number of patrols, the total distance travelled, and total number of snare observations. In addition rate encounters can be computed (how many X per km, man-hours, time). Groupings are categories such as management sectors, patrol types, patrol mandates, stations, teams, etc.

Summaries results can only be viewed in tables.
